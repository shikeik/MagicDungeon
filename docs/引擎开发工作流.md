
### 基本流程
- 如果是此版本第一次更新，改gradle.properties版本到当前版本projectVersion=x.x.x.x
- 开发更新
- 更新打包引擎核心库 generateAssetsList
- 更新打包引擎源码资源库 updateEngineLibSources
- 刷新变更历史 generateChangelog
- 刷新引擎文档网站 buildDocsDist
- 推送更新 git push

### 发布流程
- 修复缩进为tab sh indentFix.sh
- 提交修复缩进 git commit -m "chore: indentFix"
- 改gradle.properties版本projectVersion=x.x.x.x
- 刷新gradle --refresh-dependencies
- 提交 git commit
- 打标签 git tag x.x.x.x
- 推送 git push
- 推送标签 git push origin x.x.x.x
