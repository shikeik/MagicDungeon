# MagicDungeon 开发者中心

**MagicDungeon** 是一个基于 LibGDX 的现代化 2D 游戏引擎，专注于**跨平台热重载开发**与**原生级性能**。

### 核心特性
*   **双端开发：** 支持 Windows (PC) 与 Android (手机) 直接运行编辑器。
*   **ECS 架构：** 基于实体-组件-系统的高性能架构。
*   **Neon 动画：** 内置高性能骨骼动画系统。
*   **热重载：** 修改 Java 代码后，无需重启即可在编辑器内实时生效。
