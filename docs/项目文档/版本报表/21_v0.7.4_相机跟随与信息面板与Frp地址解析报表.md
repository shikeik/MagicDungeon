## 一、概述

v0.7.3 完成了同机/LAN 连接修复和大厅 UI 优化后，进入实际游戏体验阶段发现三个待改进点：

1. **相机固定在视口中心**: 坦克移出屏幕后无法看到自己，大型战场无意义。
2. **I 键抽屉面板只有成员信息**: 缺少房间连接元数据（IP、端口、连接模式等），调试不直观。
3. **Frp 地址无法携带端口**: 樱花 Frp 等工具分配的远程端口通常与本地不同（如本地 20001 → 远程 59473），大厅只能填 IP，端口被锁定为 `DEFAULT_UDP_PORT`。

本版本一次性解决以上三个问题。

## 二、问题分析与方案

### 2.1 相机跟随

**现状**: `renderPlaying()` 只使用 `uiViewport.getCamera()` 渲染一切（坦克 + HUD + 抽屉面板），相机永远在视口中心，坦克走远就看不到。

**方案**: 拆为两个渲染 Pass：

```
Pass 1 — 游戏世界 (worldCamera)
  · setProjectionMatrix(worldCamera.combined)
  · 渲染坦克、子弹
  · worldCamera 每帧跟随本地玩家坦克 (lerp 平滑)

Pass 2 — UI 固定层 (uiViewport.getCamera())
  · setProjectionMatrix(uiViewport.getCamera().combined)
  · 渲染 HUD 文字、I 键抽屉面板
  · 相机固定不动
```

跟随算法使用指数衰减插值：
```
lerp = min(1, 5 * delta)
camera.x += (tank.x - camera.x) * lerp
camera.y += (tank.y - camera.y) * lerp
```
`lerp=5` 保证 ~60fps 下约 0.2 秒追到 95% 位置，手感平滑且不会丢目标。

### 2.2 I 键面板扩展

**现状**: 抽屉面板只有「房间成员」一个区域，显示每个坦克的名称/颜色/HP/坐标。

**方案**: 在成员列表上方新增「房间信息」区，显示：

| 字段 | 示例值 |
|---|---|
| 房间名 | goldsprite 的房间 |
| 角色 | Server (房主) / Client (玩家) |
| 监听/连接 | 0.0.0.0:20001 / 119.84.246.217:59473 |
| 模式 | 回环 (同机) / LAN (局域网) / 公网/Frp |
| 坦克数 | 2 |

模式自动判断逻辑：
```
if (isServer) → "本机服务端"
if (ip == 127.0.0.1 || localhost) → "回环 (同机)"
if (ip.startsWith 192.168. / 10. / 172.) → "LAN (局域网)"
else → "公网/Frp"
```

### 2.3 Frp ip:port 地址解析

**现状**: 大厅「主机地址(可选)」输入框只作为 IP 传递，Port 被锁定为 `DEFAULT_UDP_PORT=20001`。但 Frp 分配的远程端口通常不等于本地端口。

**典型场景（樱花 Frp）**:
```
本地: 0.0.0.0:20001 (Server 监听)
         ↓ Frp 隧道
远程: 119.84.246.217:59473 (Frp 分配的公网端口)
```

**方案**: 支持 `ip:port` 格式自动解析：

```java
// 输入: "119.84.246.217:59473"
String frpIp = "119.84.246.217";
int frpPort = 59473;

// 发布到 Presence: hostIp=frpIp, hostPort=frpPort
// Server 本地仍监听: DEFAULT_UDP_PORT=20001
```

关键分离：**房间信息中的端口 = Frp 远程端口**（给客户端连），**Server 实际监听端口 = DEFAULT_UDP_PORT**（给 Frp 本地隧道连）。

## 四、核心实现细节

### 4.1 双 Pass 渲染 + 相机跟随

```java
// renderPlaying() — Pass 1: 游戏世界
neon.setProjectionMatrix(worldCamera.combined);
neon.begin();
// 渲染坦克、子弹 ...
neon.end();

// Pass 2: UI HUD (固定相机)
neon.setProjectionMatrix(uiViewport.getCamera().combined);
neon.begin();
// 渲染 HUD 文字、抽屉面板 ...
neon.end();
```

```java
// updateCameraFollow() — 每帧调用
private void updateCameraFollow(float delta) {
    TankBehaviour followTank = isServerRole ? clientTanks.get(-1) : findLocalPlayerTank();
    if (followTank != null) {
        float tx = followTank.x.getValue();
        float ty = followTank.y.getValue();
        float lerp = Math.min(1f, 5f * delta);
        worldCamera.position.x += (tx - worldCamera.position.x) * lerp;
        worldCamera.position.y += (ty - worldCamera.position.y) * lerp;
    }
    worldCamera.update();
}
```

Server 端跟踪 `clientTanks.get(-1)`（Host 坦克），Client 端跟踪 `findLocalPlayerTank()`。

### 4.3 Frp ip:port 解析

```java
// SupabaseLobbyScreen.createAndPublishRoom()
String frpIp = customIp;
int frpPort = udpPort;  // 默认 20001

if (customIp.contains(":")) {
    int colonIdx = customIp.lastIndexOf(':');
    String portStr = customIp.substring(colonIdx + 1);
    try {
        frpPort = Integer.parseInt(portStr);
        frpIp = customIp.substring(0, colonIdx);
    } catch (NumberFormatException ignored) {
        // 端口解析失败 → 整个字符串作为 IP
    }
}

// 房间信息: hostIp=frpIp, hostPort=frpPort (公网端口)
myRoomInfo = new PresenceRoomInfo(roomName, frpIp, localIp, frpPort, 1, 6);

// Server 本地监听: udpPort = DEFAULT_UDP_PORT = 20001
NetcodeTankOnlineScreen.preConfigureAsHost(udpPort);
```

关键：`preConfigureAsHost(udpPort)` 传的是本地监听端口 20001，`myRoomInfo.hostPort` 是 Frp 远程端口 59473。客户端看到并连接的是 59473。

## 五、樱花 Frp 配置指南

### 5.1 隧道创建

| 配置项 | 值 | 说明 |
|---|---|---|
| 隧道类型 | **UDP** | 游戏使用 UDP 传输 |
| 本地 IP | 192.168.31.251 | 你的局域网 IP（`ipconfig` 获取） |
| 本地端口 | **20001** | 必须与 `DEFAULT_UDP_PORT` 一致 |
| 远程端口 | 任意 | 免费节点会自动分配（如 59473） |

### 5.2 使用流程

```
1. 启动樱花 Frp 客户端 → 隧道上线
   → 日志显示: frp-put.com:59473 或 119.84.246.217:59473

2. 房主在大厅"主机地址(可选)"填入:
   119.84.246.217:59473
   （或 frp-put.com:59473，但域名可能有 DNS 延迟）

3. 点击"创建房间"
   → Server 本地监听 0.0.0.0:20001
   → 其他玩家看到房间地址: 119.84.246.217:59473

4. 其他玩家点击"加入" → 直接连接 Frp 公网端点
```

### 5.3 数据包路径

```
Client (外网)
    │
    ▼  UDP → 119.84.246.217:59473
Frp 服务端 (重庆电信 PLUS2)
    │
    ▼  Frp 隧道 → 192.168.31.251:20001
Server (房主本机)
```
