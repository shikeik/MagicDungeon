# 全局硬性规则 (Hard Rules)

> **注意**: 以下规则在任何情况下必须严格遵守，无例外。

1. **中文强制**: 
   - 所有代码注释、文档说明**必须**使用中文。
   - Git 提交信息 (`commit message`) **必须**使用中文编写。

2. **Git 操作**:
   - **禁止推送**: 严禁执行 `git push` 操作。所有变更仅执行本地提交 (`git commit`)，推送操作由人工进行。
   - **规范提交**: 必须使用 `commit_msg.txt` 模板进行规范化提交。

3. **代码规范**:
   - **禁止全限定名**: 严禁在代码中直接使用全限定名 (如 `java.util.List`)，必须使用 `import`。
