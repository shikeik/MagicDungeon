# 核心循环流程

**1. 任务启动** (Pick)
  ↓
**[判断]** 任务是否过大?
  ├─ [是] -> **1.5 任务拆解** (Decompose) -> (回到 1. 启动)
  └─ [否] -> **2. 分析与计划** (Plan) -> **3. 实现与验证** (Do & Check) -> **4. 交付与版本** (Deliver) -> **5. 清理与循环** (Loop)
       ↓
**[判断]** 任务达标或无任务?
  ├─ [否] -> (回到 1. 启动)
  └─ [是] -> **收尾统计** -> **结束**
