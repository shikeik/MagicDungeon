# 第五阶段：清理与循环 (Loop)

1.  **清理交付**:
    - 移除临时 Log 和调试代码。
    - 更新相关文档。
    - **记录踩坑**: 若有价值，记录到 `.trae/documents/pitfalls`。
2.  **状态更新**:
    - 在 `Tasks.md` 中标记当前任务为 `[x] (待检查)` 并统一移动到下方待检查区
    - 更新 `[当前已完成]` 计数 (+1)。
3.  **循环决策**:
    - **未达标/有任务**: -> **回到第一阶段 (智能拾取)**。
    - **已达标/无任务**: -> **执行收尾**:
        1. 在 `Tasks.md` 顶部记录**完成时间**、**总耗时**、**实际完成数**, 精确到分钟。
        2. 重置 `[当前已完成]` 为 0。
        3. **结束流程**。
