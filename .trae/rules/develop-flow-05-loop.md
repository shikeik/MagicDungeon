# 第五阶段：清理与循环 (Loop)

1.  **清理交付**:
    - 移除临时 Log 和调试代码。
    - 更新相关文档。
    - **记录踩坑**: 若有价值，记录到 `.trae/documents/pitfalls`。
2.  **状态更新**:
    - 在 `Tasks.md` 中标记当前任务为 `[x]` 并统一移动到下方待检查区
    - 在 `Tasks.md` 顶部记录**当前完成时间**、**当前已耗时**、**当前已完成数**, 精确到分钟。
3.  **循环决策 (重要: 必须自动执行)**:
    - **未达标 (当前已完成 < 目标拾取数) 且 有剩余任务**: 
        - **严禁停止**: 不要询问用户，不要结束 Turn。
        - **立即执行**: 直接调用 `Read` 工具读取 `develop-flow-01-pick.md`，并**自动开始下一轮循环**。
    - **已达标 或 无剩余任务**: -> **执行收尾**:
        1. 重置 `[当前已完成]` 为 0。
        2. **结束流程**。
